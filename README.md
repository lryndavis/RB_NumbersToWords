# Numbers to Words 

By Lauryn Davis + Jeff Seymour 

A Ruby/Sinatra app translates numbers in numeric form into written words. 

####Technologies Used

* Ruby
* Sinatra
* rspec
* HTML
* CSS

#### Setup

* Clone this repository
* Navigate to the project directory in the command line and run:
`bundle install`
`ruby app.rb`
